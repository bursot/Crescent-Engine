# Contributing

Thanks for taking the time to contribute.

## Setup
- Run `./scripts/setup_deps.sh` to fetch and build dependencies.
- Open `CrescentEngine.xcodeproj` in Xcode.

## Pull Requests
- Keep changes focused and scoped.
- Describe the intent and any tradeoffs.
- If you add features, include a short test or a minimal repro.

## Issues
- Include OS version, Xcode version, and repro steps.
